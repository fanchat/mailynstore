# Mailyn.cn — 使用指南

> 给你的女儿。这是一个集商城 + 社交圈子于一体的独立网站。
> 爸爸在惠州为你搭的。

---

访问网站：

```
http://mailyn.cn:8000
```
> 网站会自动定向到 `/dk/`（丹麦区域页面），地址栏会显示 `http://mailyn.cn:8000/dk/`
> 目前是测试阶段，还没配 HTTPS，以后会加上。
> 在德国如果打不开，试试在地址栏手动加 `https://` 或直接用 Chrome。

## 二、注册与登录

### 注册新账号

```
首页 → 右上角"账户" → 底部"注册" → 填邮箱+密码
```

关键点：
- **邮箱**就是你的登录名，也是别人加你好友的标识
- **密码**设置好记的就行
- 注册成功自动登录，可以直接用

### 登录

```
首页 → 右上角"账户" → 输入邮箱和密码
```

### 退出

```
首页 → 右上角"账户" → 个人中心 → 找退出/注销按钮
```
（或者关掉浏览器标签页即可，下次打开需要重新登录）

## 三、商城购物

网站首页就是商城入口，可以看到商品和分类。

| 页面 | 路径 |
|------|------|
| 首页/商品展示 | `http://mailyn.cn:8000` |
| 分类浏览 | 首页点"分类"或"全部商品" |
| 购物车 | 右上角购物车图标 |
| 结算下单 | 购物车 → 结算 |

> 目前支付还没对接（Stripe），下单后会生成订单但不会扣款。

## 四、社交功能（核心）

社交功能在网站里是一个独立的板块，点底部导航栏的 **"圈子"** 进入。

### 底部导航栏

| 图标 | 名称 | 说明 |
|------|------|------|
| 🏪 | **商城** | 回到首页买买买 |
| 💬 | **消息** | 好友私信（有红点=未读） |
| 🌐 | **圈子** | 朋友圈/动态 |
| 🔍 | **搜索** | 找人、找服务、找商品 |
| 👤 | **我的** | 个人资料、私密帖子 |

### 动态 / Feed（圈子）

进入"圈子"可以看到所有公开的动态：
- **点赞** — 点❤️
- **评论** — 点💬输入
- **发帖** — 点右下角"+"按钮

发帖时可以：
- 输入文字
- 上传图片/视频
- 选择可见范围：**公开** / **好友可见** / **仅自己**

> 发帖时系统会自动生成三份——个人版、好友版、工作版，你可以分别编辑。

### 好友

```
圈子 → 底部"好友"标签
```
- **搜索加好友** — 输入对方的邮箱（就是她的登录邮箱）搜索 → 发送好友请求
- **好友列表** — 已加的好友都在这里
- **删除好友** — 好友列表里操作

### 私信聊天

有**两种方式**进入聊天：

**方式一：** 好友列表 → 点好友名字 → 弹出"发消息"按钮
**方式二：** 底部"消息"图标 → 看到所有有过聊天的好友列表 → 点进去

聊天功能：
- 发送文字消息
- 发送图片（支持拍照或相册选图）
- 新消息有**红色未读标记**

### 个人主页

```
圈子 → 底部"我的"
```

这里显示：
- 你的头像、昵称、签名
- 你发过的**私密帖子**（仅自己可见）
- **改名**、**改头像**、**改签名**
